# React Fundamental

## Description

This is my fifteenth task for the Alterra program, and here I am continuing my React todo project. The difference is that I'm migrating from local state to global state using Redux.

## Technology

- React with the CRA Template

- Bootstrap

## How to Install

1. Clone this repository into your local directory.
2. Run the `npm install` command to install dependencies for the project.

## Run Aplication

1. Run the `npm start` command to run the app in development mode.
2. Open [http://localhost:3000](http://localhost:3000) in the browser to view the application.
