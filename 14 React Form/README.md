# React Fundamental

## Description

This is my fourteenth assignment from the Alterra program, where I created a registration form that needs to be validated before it can be submitted. Once the validation is successful and there are no errors, a notification will appear stating that the data has been successfully received.

## Technology

- React with the CRA Template

- Bootstrap

## How to Install

1. Clone this repository into your local directory.
2. Run the `npm install` command to install dependencies for the project.

## Run Aplication

1. Run the `npm start` command to run the app in development mode.
2. Open [http://localhost:3000](http://localhost:3000) in the browser to view the application.
